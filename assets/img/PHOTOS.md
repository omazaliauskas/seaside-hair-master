# Nuotraukų aplankas — įdėkite failus čia

Įkelkite savo nuotraukas į šį aplanką (`assets/img/`) tiksliai šiais pavadinimais.
Kai tik failai bus čia, jie automatiškai atsiras svetainėje (galerijoje, „Prieš/Po“ kortelėse ir kategorijų puslapiuose).

Formatas: **JPG** (rekomenduojama) arba PNG. Idealus plotis ~1200–1600 px.

| Failo pavadinimas        | Kokią nuotrauką įdėti (pagal jūsų atsiųstas) |
|--------------------------|-----------------------------------------------|
| `blonde-bob.jpg`         | Šviesus tiesus „bob“ kirpimas, vaizdas iš nugaros |
| `styling-dryer.jpg`      | Ranka su plaukų džiovintuvu, ilgi rudi plaukai |
| `long-brown.jpg`         | Ilgi tiesūs šviesiai rudi plaukai (laiptai fone) |
| `stylist-blowdry.jpg`    | Stilistė žalsvu švarku pučia klientės plaukus |
| `balayage-wash.jpg`      | Karamelinis balajažas prie plovimo kriauklės |
| `long-ombre.jpg`         | Ilgi karamelinio/ombre atspalvio plaukai (laiptai) |
| `foils-highlights.jpg`   | Stilistė daro sruogeles su folijomis (mėlynas sijonas) |
| `blonde-closeup.jpg`     | Šviesių sruogelių stambus planas iš šono |

> Norite pridėti daugiau darbų? Įkelkite papildomus failus ir pridėkite juos į
> `js/gallery-data.js` (masyvas `works`). Kiekvienas įrašas turi `img`, `title` ir `cats`.
